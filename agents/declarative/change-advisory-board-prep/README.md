# Change Advisory Board Prep — Deployment Guide

## Overview
Declarative agent grounded in the org's change management docs. Helps requestors draft RFCs and helps CAB chairs prepare briefings. No Graph API needed.

**Time to deploy:** ~15 minutes | **License:** M365 Copilot

## Prerequisites
- [ ] M365 Copilot license
- [ ] SharePoint site with: change management policy, RFC template, change calendar

## Deployment
1. **Update manifest.json** — Set SharePoint URL to change management site.
2. **Package & Upload** → Teams Admin Center.
3. **Access** — IT operations, platform engineering, change management teams.
4. **Test** — Ask: *"Help me write a CAB submission for upgrading the Intune connector this weekend."*
