# Policy-to-Enforcement Mapper — Deployment Guide
## Overview
Copilot Studio agent mapping documented policies to actual M365 enforcement controls.
**Architecture:** Copilot Studio | **Approval:** No | **License:** M365 Copilot
## Deployment
1. App registration: `Policy.Read.All`, `DeviceManagementConfiguration.Read.All`, `InformationProtectionPolicy.Read`.
2. SharePoint policy library with documented security policies.
3. Create bot with topics: Full gap analysis, Single policy mapper, Unenforced policies.
4. Publish — compliance and security teams.
